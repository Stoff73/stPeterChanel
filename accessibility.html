Accessibility

Home/Accessibility

We are committed to making our website accessible to everyone, including people with disabilities.

Accessibility Statement

This accessibility statement applies to the St Peter Chanel Catholic Primary School website. We want as many people as possible to be able to use this website. This means you should be able to:

  • Change colours, contrast levels and fonts
  • Zoom in up to 300% without the text spilling off the screen
  • Navigate most of the website using just a keyboard
  • Navigate most of the website using speech recognition software
  • Listen to most of the website using a screen reader

We have also made the website text as simple as possible to understand.

Accessibility Features

Our website includes the following accessibility features:

High Contrast Mode

Use the contrast button in the top bar to switch to a high contrast colour scheme for improved readability.

Keyboard Navigation

All interactive elements can be accessed and operated using only a keyboard.

Skip Links

Skip to main content links allow keyboard users to bypass navigation and go straight to the main content.

Translation

Google Translate is available to translate the website into different languages.

Responsive Design

The website works on all devices and screen sizes, from mobile phones to desktop computers.

Text Resizing

Text can be resized using your browser's zoom function without losing functionality.

Standards Compliance

We aim to meet the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standard. This website has been designed with the following in mind:

  • Perceivable: Information is presented in ways that users can perceive
  • Operable: User interface components are operable
  • Understandable: Information and operation of the interface is understandable
  • Robust: Content is robust enough to be interpreted by a wide variety of user agents

Technical Information

This website is designed to be compatible with the following assistive technologies:

  • Screen readers (JAWS, NVDA, VoiceOver)
  • Screen magnifiers
  • Speech recognition software
  • Keyboard-only navigation

Browser Compatibility

This website is designed to be compatible with recent versions of the following browsers:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Microsoft Edge

Known Issues

We are aware of the following accessibility issues and are working to resolve them:

  • Some PDF documents may not be fully accessible. If you need information from a PDF in an alternative format, please contact us.
  • Some embedded content from third-party services may not be fully accessible.

How to Get Help

If you have difficulty accessing any part of our website, please contact us and we will provide the information you need in an alternative format.

Contact Us About Accessibility

If you find any accessibility issues or need information in a different format, please contact us:

  • 020 8300 7280
  • admin@st-peterchanel.bexley.sch.uk

We will respond to your request within 5 working days.

Enforcement Procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018. If you're not happy with how we respond to your complaint, you can contact the Equality Advisory and Support Service (EASS).
